# 📈 Market Score Monitor

主要な市場データを毎朝かんたんに確認し、**日経225マイクロ取引の参考用**に「市場環境スコア」を表示する軽量 Web アプリです。
Streamlit 製で、GitHub + Streamlit Community Cloud だけで無料公開できます。

> ⚠️ 本アプリは投資判断の **参考情報** を提供するものであり、特定の売買を推奨するものではありません。最終的な取引判断はご自身の責任で行ってください。

---

## 主な機能

- NASDAQ / S&P500 / SOX / NVIDIA / USDJPY / 日経平均 / 日経先物 などの市場データを取得
- 各項目を独自ルールで OK / NG 判定し、合計を **総合スコア（0〜10点）** として表示
- スコアに応じて「市場環境」「信頼度」「参考ロット」を表示
- 「最新データ取得」ボタンで再取得（押下時はキャッシュをクリアして最新値を反映）

---

## 画面に表示される指標

| 指標 | 内容 |
| --- | --- |
| 総合スコア | OK 項目の合計点（最大 10 点） |
| 市場環境 | スコアを「警戒／中立／やや強い／強い／非常に強い」で表現 |
| 信頼度 | 総合スコア ÷ 満点 の割合（%） |
| 参考ロット | スコアに対応したリスク確認用の目安枚数 |

### スコアと評価の対応

| スコア | 市場環境 | 参考ロット |
| --- | --- | --- |
| 0〜3 | 警戒 | 0 枚 |
| 4〜5 | 中立 | 1 枚 |
| 6〜7 | やや強い | 2 枚 |
| 8〜9 | 強い | 3 枚 |
| 10 | 非常に強い | 5 枚 |

> 参考ロットはあくまでリスク確認のための目安です。実際の建玉数を保証・推奨するものではありません。

## ローカルでの実行

Python 3.9 以上を推奨します。

```bash
# 1. リポジトリを取得
git clone https://github.com/<your-account>/<your-repo>.git
cd <your-repo>

# 2. 依存パッケージをインストール
pip install -r requirements.txt

# 3. アプリを起動
streamlit run app.py
```

起動後、ブラウザで `http://localhost:8501` を開きます。

## Streamlit Community Cloud での公開手順

1. このリポジトリを GitHub に push する（`app.py` / `requirements.txt` / `README.md` を含める）
2. [share.streamlit.io](https://share.streamlit.io) にアクセスし、GitHub アカウントでログイン
3. 「New app」→ 対象リポジトリ・ブランチ・`app.py` を選択
4. 「Deploy」をクリック

数十秒〜数分でビルドが完了し、公開 URL が発行されます。

## 注意事項・既知の制限

- **データソースについて**: 株価・指数データは [yfinance](https://github.com/ranaroussi/yfinance) 経由で取得しています。提供元の都合でデータが取得できない場合、該当項目は「未対応」と表示されます（特に `^N225` などの指数系は一時的に取得できないことがあります）。
- **「毎朝自動取得」について**: アプリはページを開いた瞬間にデータを取得します。Streamlit Community Cloud は一定時間アクセスがないとスリープするため、文字どおり無人で毎朝更新したい場合は、外部の定期実行サービス（cron、UptimeRobot 等）で URL に定期アクセスさせる必要があります。
- **バージョン固定について**: yfinance は更新で仕様が変わることがあります。安定運用したい場合は `requirements.txt` でバージョンを固定（例：`yfinance==0.2.x`）してください。
